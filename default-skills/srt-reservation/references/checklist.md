# SRT Intake Checklist

Use this compact template when the user gives only a partial booking request.

```text
SRT booking details
- One-way or round-trip:
- Departure station:
- Arrival station:
- Date:
- Time window:
- Passengers:
- Seat class:
- Flexible to earlier/later trains:
- Waitlist acceptable:
```

When enough fields are already known, ask only for the missing lines instead of repeating the full template.
